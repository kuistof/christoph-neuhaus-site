Eskalationsmechanik

🔥 4-Stunden-Sprint

Wenn der Customer Success Manager ein Problem erkennt, wird die gesamte Agentur für 4 Stunden auf dieses eine Problem geblockt. Kein Multitasking, kein „machen wir nächste Woche".

Trigger
CSM erkennt Problem → meldet es → Sprint wird für nächsten verfügbaren Slot geblockt
Wer arbeitet?
Alle relevanten Rollen: Stratege, Copywriter, Designer, Editor — zusammen an einem Tisch
Output
Neue Ad-Strategien, frische Creatives, Roadmap, klare Verantwortlichkeiten — alles in 4h
Ergebnis
CSM geht ins nächste Kundenmeeting mit konkreter Lösung → Zufriedenheit wiederhergestellt
Der Zaubertrick

✦ Individuelle Kunden-HTML nach Sprint-Session

Nach der initialen Sprint-Session bekommt der Kunde eine individuelle URL mit einer Seite in seinem Branding. Zusammenfassung, Roadmap, KPIs — der „Wow, die haben's verstanden"-Moment.

01
Setting Call + Marktrecherche
Input sammeln: Was macht die Firma, Branche, Ziele, bisherige Versuche
02
KI-Agent füllt Template
Automatisiert: Template + Call-Notizen + Branchendaten → 90% vorgefüllt KI ~90%
03
Menschlicher Feinschliff
Stratege prüft, ergänzt individuelle Insights, schärft Roadmap
04
HTML im Kundenbranding
Claude generiert Seite mit Farben/Logo des Kunden → Subdomain oder URL
